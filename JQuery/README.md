# JQuery

## Course Description
This directory contains labs and resources covering concepts in **JQuery**.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab](Lab/) | **Core jQuery Interactions and AJAX** | Practical exercises covering DOM manipulation, events, animation effects, dynamic element creation, event delegation, and consuming JSON data using AJAX |

## Learning Outcomes

After completing this folder, you should be able to:

1. Select and manipulate DOM elements using jQuery selectors and helpers.
2. Handle user interactions with click, hover, and delegated event handlers.
3. Create simple interactive UI components (menu, slider, to-do list).
4. Consume external APIs with `$.ajax()` and render returned data.
5. Structure small front-end tasks into reusable, readable scripts.

## Practice Focus

- **Interactivity**: respond to user actions with instant visual feedback.
- **Dynamic UI updates**: create/remove elements without page reload.
- **Data rendering**: transform JSON API responses into HTML tables.
