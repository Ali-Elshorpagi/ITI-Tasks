# Responsive Web Fundamentals

## Course Description

This directory contains labs designed to solidify the core principles of **Responsive Web Design (RWD)**. The projects focus on scaling web user interfaces beautifully across all screen dimensions using native CSS tools as well as modern CSS frameworks.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **Bootstrap 5 Elements** | Constructing a fully-fledged Social Media Dashboard utilizing Bootstrap's 12-column grid system and responsive utility classes. |
| [Lab02](Lab02/) | **Responsive UI Components** | Recreating specific interface panels (Drag & Drop Survey Builder, Employee Directory) using native scalable sizing units and modern boundaries. |
| [Lab03](Lab03/) | **Single Page Portfolio** | Developing a complete responsive portfolio website incorporating navigation toggles, adaptive grids, and structured media queries. |
