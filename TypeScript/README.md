# TypeScript

## Course Description
This directory contains practical labs for learning **TypeScript fundamentals** through small, focused exercises.

The labs introduce core ideas such as interfaces, optional properties, classes, type-safe function design, and DOM event handling with null checks.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **Interfaces and Typed Data Processing** | Builds a student reporting workflow using a typed interface, array operations, and status classification based on average grades |
| [Lab02](Lab02/) | **Class-Based DOM Counter App** | Implements an interactive counter using a class with increment/decrement/reset behavior and event-driven DOM updates |

## Learning Outcomes

After completing these labs, you should be able to:

1. Define and use interfaces to model real-world data.
2. Write typed functions with clear return types and logic boundaries.
3. Apply array methods (`reduce`, `forEach`) in type-safe workflows.
4. Build simple classes to encapsulate state and behavior.
5. Connect TypeScript logic to browser UI with safe DOM handling.

## Practice Focus

- **Type safety**: preventing common runtime mistakes by design.
- **Readability**: structuring logic into reusable functions and classes.
- **Browser integration**: updating UI elements based on user actions.
