# JavaScript Basics

## Course Description

This directory contains labs and resources covering concepts in **JavaScript Basics** as part of the Client Side Technologies track.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **JS Fundamentals** | Variables, `prompt`, `parseInt`/`parseFloat`, `toFixed`, `NaN` check, input validation, external vs internal scripts |
| [Lab02](Lab02/) | **Arrays & Strings** | Remove duplicates, sort, filter > 50, max/min, sum/product, `eval()`, capitalize first letter, find longest word |
| [Lab03](Lab03/) | **BOM & Timers** | `window.open()`, `setInterval`/`clearInterval`, URL query params (`URLSearchParams`), login validation with child windows |
| [Lab04](Lab04/) | **DOM Manipulation & Events** | Selectors (`getElementById`, `querySelector`), DOM traversal, image slideshow, student dropdown, dynamic number grid, search/sort table |
| [Lab05](Lab05/) | **ElectroShop — LocalStorage App** | Multi-page e-shop with product cards, search, Add to Cart/Favorites using `localStorage`, cart with quantity management |
