# HTML5 and CSS3

## Course Description

This directory contains labs focused on core concepts in **Modern Web Development**, specifically leveraging the new foundational features of **HTML5** and the advanced structural capabilities of **CSS3**.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **HTML5 APIs & Elements** | Integration of HTML5 APIs including Geolocation with Google Maps, `<canvas>` drawing tools, and custom multimedia `<audio>` controllers. |
| [Lab02](Lab02/) | **CSS Flexbox** | Structuring responsive web layouts utilizing the Flexible Box Module (Flexbox) to handle alignment and content distribution. |
| [Lab03](Lab03/) | **CSS Grid** | Building complex two-dimensional layout systems (dashboards) relying strictly on CSS Grid architecture. |
