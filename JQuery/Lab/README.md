# jQuery Lab

## Lab Description
This lab contains four focused jQuery mini-projects that build practical front-end skills through hands-on interaction patterns.

## Exercises

| Exercise | Topic | Description |
|----------|-------|-------------|
| [ajax_json](ajax_json/) | **AJAX and JSON Rendering** | Fetches post data from an external API and renders it into an HTML table dynamically |
| [slider](slider/) | **Image Slider and Lightbox Behavior** | Displays selected images in an overlay slider with next/previous navigation |
| [slide_menu](slide_menu/) | **Expandable Menu Interactions** | Uses hover and click events to expand a menu and toggle sub-items |
| [todo_list](todo_list/) | **Dynamic To-Do List** | Adds and removes tasks in real time using DOM creation and event delegation |

## Learning Objectives
- Practice selecting elements and binding event handlers with jQuery
- Build UI behavior with hover, click, fade, and class toggling
- Dynamically generate and remove HTML elements based on user actions
- Read API responses and map JSON fields to table rows
- Organize mini features into clear, maintainable scripts

## Typical Workflow
1. Open an exercise page in the browser.
2. Trigger user actions (click, hover, add/remove).
3. Inspect DOM updates and rendered output.
4. Refine script logic and verify edge cases.
