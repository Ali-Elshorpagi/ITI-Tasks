# C++ Basics and OOP

## Course Description

This directory contains labs and resources covering concepts in **C++ Basics and Object-Oriented Programming**.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **C++ Fundamentals** | ASCII conversions, arithmetic calculator, character case conversion |
| [Lab02](Lab02/) | **Intermediate Algorithms** | Prime check, grade classifier, factorial, fast exponentiation, number reverse, binary conversion, magic square |
| [Lab03](Lab03/) | **Arrays & Strings** | 1D array operations (max, min, sum, search, bubble sort), 2D matrix (row sums, column averages), string concatenation, case conversion |
| [Lab04](Lab04/) | **Menu-Driven TUI** | Keyboard-navigable console menu with name manager, arithmetic, factorial, power, string length/copy, Fibonacci |
| [Lab05](Lab05/) | **Structs & Pointers** | Employee management system using `struct`, dynamic allocation (`new`/`delete`), soft deletion, pointer-based swap |
| [Lab06](Lab06/) | **Dynamic Memory & Line Editor** | `Employee` class, 1D/2D dynamic arrays, interactive single-line text editor with cursor navigation and case toggle |
| [Lab07](Lab07/) | **OOP Foundations** | `Complex` and `Employee` classes, constructor overloading, destructors, pass-by-value vs pointer vs reference |
| [Lab08](Lab08/) | **Stack Implementations** | `StaticStack` (fixed array) vs `DynamicStack` (heap), deep copy constructor, pass-by comparison tests |
| [Lab09](Lab09/) | **Composition vs Aggregation** | `Point`, `Rectangle`, `Triangle` classes demonstrating ownership by value (composition) vs by pointer (aggregation) |
| [Lab10](Lab10/) | **Templates & Inheritance** | Template `Stack<type>`, `Swap<type>()`, `Person‚ÜíEmployee‚ÜíInstructor` hierarchy, full CRUD system with bubble sort |
| [Lab11](Lab11/) | **Polymorphism** | Abstract `Shape` class with pure virtual `getArea()`, `Rectangle`, `Square`, `Triangle`, `Circle` subclasses, polymorphic area calculator |
| [Lab12](Lab12/) | **Operator Overloading (Stack)** | Template `Stack` with overloaded `=`, `+` (concatenation), and `friend operator<<` for stream output |
| [Lab13](Lab13/) | **Operator Overloading (Fraction)** | `Fraction` class with `+`, `==`, `!=`, `<`, `>`, `++`, `=`, `<<`, `operator float()`, static counter, GCD auto-simplification |
