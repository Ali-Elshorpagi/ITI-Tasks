# Lab03 — Advanced OOP and Architecture Patterns

## Lab Description

This lab dives into more advanced Object-Oriented Programming (OOP) architectures in **C#**. It covers cloning techniques, operator overloading, and static classes, alongside widely-used design patterns.

## Topics Covered

- Building robust custom classes (e.g., `Point3D`) implementing standard interfaces like `IComparable<T>` and `ICloneable`.
- Operator overloading for mathematical equality (`==`, `!=`) and explicit casting (`explicit operator string`).
- Utilizing static classes and `Math` helper methods.
- Designing a Singleton architectural pattern for restricted instantiations (`NIC` network interface controller).

## Projects

| Project | Description |
|---------|-------------|
| **Task01** | Demonstrates `Point3D` structure, custom equality checks (overriding `Equals` & `GetHashCode`), operator overloads, and explicit string casting. |
| **Task02** | Handles static classes and extension/math methods mapping basic algebraic rules. |
| **Task03** | Contains an implementation of the Singleton pattern (ensuring only a single instance of a class ever exists). |
