# HTML & CSS

## Course Description

This directory contains labs and resources covering concepts in **HTML & CSS** as part of the Client Side Technologies track.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **HTML Fundamentals** | Font tags, preformatted text, images with text wrapping, nested ordered lists, image maps, and a restaurant page layout |
| [Lab02](Lab02/) | **Tables & Forms** | Hyperlink types (SMS, tel, WhatsApp, download), complex tables with `colspan`/`rowspan`, nested tables, timetable layout, and a registration form |
| [Lab03](Lab03/) | **Multi-Page Website with CSS** | A complete e-commerce-style website with Products, About Us, Contact Us, Login, Sign Up, and Product detail pages — all styled with external CSS |
