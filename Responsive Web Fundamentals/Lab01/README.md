# Lab01

## Overview

This lab focuses on rapid responsive prototyping utilizing the **Bootstrap 5** framework. It involves actively translating a static design into a fully functional Social Media Profile layout that gracefully collapses across varied device widths.

## Tasks / Features

### Social Media Dashboard 
- **Macro Layout**: Evaluates Bootstrap's structural classes (`container-fluid`, `row`), carving the layout into a classic 3-column feed.
- **Responsive Columns**: Allocates spatial rules depending on the viewport using breakpoints like `col-lg-2`, `col-md-4`, and `col-lg-6` to ensure the sidebar collapses intuitively or expands optimally on wide monitors.
- **Component Styling**: Leverages Bootstrap component abstractions like Cards (`card`, `card-body`), Navigation Bars (`navbar`, `nav-item`), and Badges.
- **Custom Enhancements**: Layers custom CSS over Bootstrap's base styles to fine-tune alignments, custom background colors (`bg-group`), and borders efficiently.

---

## Key Concepts Demonstrated

- **Bootstrap 5 Framework Paradigm** — Integrating via CDN and applying standardized utility classes.
- **The 12-Column Grid System** — Calculating spatial allocations mathematically.
- **Breakpoint Logic** — Adapting design conditionally (`sm`, `md`, `lg`) without writing explicit CSS media queries.
- **Flex Utilities** — Modifying spacing and axis alignment natively through Bootstrap variables (`d-flex`, `me-2`, `ms-auto`).
