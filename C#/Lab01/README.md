# Lab01 — Basic Logic and Algorithms

## Lab Description

This lab focuses on fundamental programming concepts in **C#**, covering basic I/O, array manipulation, string parsing, and algorithm complexity optimizations.

## Topics Covered

- Taking input from the console and manipulating strings (splitting and reversing).
- Using loops and nested loops to find patterns in arrays (maximum distance between two identical numbers).
- Algorithm optimization: comparing string counting methods vs. mathematical (modulus) and formula-based approaches for heavy computational tasks (counting occurrences of '1' optimally in `O(log N)`).

## Projects

| Project | Description |
|---------|-------------|
| **Task01** | Reads a sentence, splits it, and prints the words in reverse order using `Array.Reverse()`. |
| **Task02** | Finds the maximum distance between two identical integer values in an array. |
| **Task03** | Demonstrates the performance gap (using `Stopwatch`) between different ways of counting a specific digit across millions of numbers. |
