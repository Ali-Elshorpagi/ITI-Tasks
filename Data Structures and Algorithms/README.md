# Data Structures and Algorithms

## Course Description

This directory contains labs and problem-solving solutions spanning core concepts in **Data Structures and Algorithms** implemented in C++. The content explores fundamental data structures (Linked Lists, Stacks, Queues) and classical algorithms (Searching, Sorting).

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **Linked Lists** | Implementation of a generic Doubly Linked List and its integration in an Employee CRUD application system. |
| [Lab02](Lab02/) | **Stacks & Queues** | Array-based Circular Queue, Linked-List based Stacks and Queues, and expression evaluation. |
| [Lab03](Lab03/) | **Searching Algorithms** | Implementation of searching techniques like Binary Search (Iterative and Recursive). |
| [Lab04](Lab04/) | **Sorting Algorithms** | Array manipulation utilizing key sorting algorithms: Bubble, Insertion, Selection, and Quick Sort. |
| [PS](PS/)      | **Problem Solving** | Solutions to various LeetCode problems heavily focused on algorithmic thinking and data structure manipulation. |
