# Lab02 — Structs, Enums, and Arrays

## Lab Description

This lab introduces custom value types in **C#** using `struct` and `enum`. It focuses on creating structured, memory-efficient configurations for entity representations.

## Topics Covered

- Declaring and utilizing custom `enum` types (e.g., `SecurityLevel`, `Gender`).
- Creating structures (`struct`), defining custom constructors, and overriding foundational methods like `ToString()`.
- Building comprehensive entity tracking mechanisms like an Employee Input System.
- Sorting arrays of complex objects using lambda functions and underlying interfaces (`IComparable`).

## Projects

| Project | Description |
|---------|-------------|
| **Task01** | Develops an Employee Entry System utilizing custom `Employee` struct, `HireDate` struct, and Enums for Security and Gender. Also implements helper menus and sorts the stored data chronologically. |
