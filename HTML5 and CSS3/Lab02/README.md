# Lab02

## Overview

A styling implementation lab heavily focused on structuring 1D container dimensions using the **CSS Flexible Box Layout (Flexbox)**. The goal is to build a modern visual skeleton dynamically adapting to internal content contexts.

## Tasks / Features

### CSS Flexbox Implementation
- **Navigation Layouts**: Horizontally dispersing navigation buttons evenly across the header bar.
- **Content Blocks**: Creating vertically or horizontally segmented primary content cards adjusting automatically based on screen widths alongside embedded footers.
- **Feature Displays**: Stacking and structuring inline feature nodes containing identical sub-elements (circular icons, bold titles, and subtitles) perfectly spaced out at the bottom of the page.

---

## Key Concepts Demonstrated

- **Flex Containers** — Declaring `display: flex;` mapping parents.
- **Axis Alignment** — Operating `justify-content` (main axis spacing) and `align-items` (cross axis bounding) properly.
- **Responsive Wraps** — Bounding continuous strings using `flex-wrap` mechanisms.
- **Semantic Tags** — Using proper HTML tags like `<nav>` and `<footer>` rather than standard `divs`.
