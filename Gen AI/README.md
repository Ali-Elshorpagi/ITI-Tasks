# Vibe Coding

## Course Description
This directory contains labs and projects for **Vibe Coding**, focusing on practical AI-assisted development workflows, full-stack implementation, and modern web app engineering.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **Algorithms, Testing, and API Integration** | QuickSort project with browser UI, optimized sorting implementation, benchmarks, unit tests, and REST API endpoint. |
| [Lab02](Lab02/) | **AI-Powered Full-Stack Application** | ChatGPT-style UI clone using Gemini with optional Hugging Face and OpenAI integrations for text, image, STT, and TTS features. |
