# EF Core

## Course Description
This directory contains labs and resources covering concepts in **EF Core**.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **Book Catalog Data Model** | Build a realistic publishing domain (books, authors, tags, reviews, and price offers), configure relationships with Fluent API, and generate the first migration to create the database schema |
| [Lab02](Lab02/) | **Layered Architecture and Data Operations** | Organize an EF Core solution into Domain, Infrastructure, and Application projects; configure entities via separate configuration classes; then practice CRUD operations and loading strategies |

## Learning Path

This EF Core track is designed to move from **core modeling skills** to **clean architecture and data access workflows**:

1. **Lab01** establishes entity design, navigation properties, and relationship mapping.
2. **Lab02** applies those foundations in a layered solution with clearer separation of concerns and maintainable configuration.

By the end of these labs, you should be comfortable creating DbContexts, mapping entities with Fluent API, generating migrations, and reading/updating related data efficiently.
