# Lab01

## Overview

A collection of **six standalone HTML pages** introducing fundamental HTML concepts: **font styling**, **preformatted code blocks**, **image alignment with text**, **nested ordered lists**, **image maps**, and a **restaurant page layout**.

## Assignment Details

### 1. Font Tag Basics (`assignment01.html`)

- Demonstrates the legacy `<font>` tag with its attributes:
  - `size` — controls text size
  - `color` — sets text color (e.g. `orange`)
  - `face` — changes font family (e.g. `INK Free`)
- Shows bold text using `<b>` and paragraphs using `<p>`.

---

### 2. Preformatted Code Display (`assignment02.html`)

- Uses `<pre>` and `<p>` to display a **C program** (recursive exponentiation) with preserved whitespace and indentation.
- Demonstrates HTML character entities: `&lt;` and `&gt;` for angle brackets.

---

### 3. Image with Text Wrapping (`assignment03.html`)

- Recreates "Ricardo's Page" — a personal page with:
  - An **image aligned left** (`align="left"`) with paragraph text wrapping around it.
  - A horizontal rule (`<hr>`) with color.
  - Styled links using `<font>` with color and underline (`<u>`).

---

### 4. Nested Ordered Lists (`assignment04.html`)

- Builds a **three-level nested ordered list**:
  - **Level 1**: Fruits, Vegetables
  - **Level 2**: Mango, Orange / Cabbage, Capsicum
  - **Level 3**: Green, Yellow, Red Capsicum
- Demonstrates `<ol>` nesting for hierarchical data.

---

### 5. Image Map (`assignment05.html`)

- Creates a **clickable image map** using `<map>` and `<area>`:
  - `shape="circle"` → links to Google
  - `shape="rect"` → links to YouTube
  - `shape="poly"` → links to ChatGPT
- Demonstrates `usemap` attribute on `<img>`.

---

### 6. Restaurant Page Layout (`assignment06.html`)

- Builds a **"Black Goose Bistro"** restaurant page with:
  - A centered header image and title.
  - Colored horizontal rule separator.
  - Sections: **The Restaurant**, **Catering**, **Location and Hours**.
  - Uses `<font>` for red section headers and `<i>` for italicized content.

---

## Key Concepts Demonstrated

- **Legacy font styling** — `<font>` tag with `size`, `color`, `face` attributes
- **Preformatted text** — `<pre>` for preserving whitespace
- **Image alignment** — `align="left"` for text wrapping around images
- **Nested lists** — multi-level `<ol>` nesting
- **Image maps** — `<map>`, `<area>` with `circle`, `rect`, `poly` shapes
- **HTML entities** — `&lt;`, `&gt;` for special characters
- **Basic page structure** — headings, paragraphs, horizontal rules, line breaks
