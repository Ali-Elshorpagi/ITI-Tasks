# C#

## Course Description
This directory contains labs and resources covering core concepts in **C#** programming. The topics progress from procedural fundamentals to advanced language paradigms handling robust OOP logic, memory management, generic structuring, asynchronous threading capabilities naturally complementing modern full-stack workflows.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **Basic Logic & Algorithms** | Manipulating native iteration, parsing strings contextually, and mapping algorithm efficiency via generic mathematical reductions. |
| [Lab02](Lab02/) | **Structs, Enums & Arrays** | Managing custom Value-Types, grouping constant states systematically, sorting structural objects safely. |
| [Lab03](Lab03/) | **Advanced OOP & Patterns** | Overloading logical operators securely, executing implicit casting internally, maintaining internal singletons efficiently mapping static instances. |
| [Lab04](Lab04/) | **Generics & Complex Types** | Declaring highly optimized strong-types dynamically bypassing explicit object boxing routing Custom Dynamic Generic Lists correctly. |
| [Lab05](Lab05/) | **Delegates, Funcs & Lambdas** | Routing behavior patterns sequentially injecting external delegates directly substituting explicit iteration filters utilizing Arrow syntax functions securely. |
| [Lab06](Lab06/) | **Events & Architectures** | Resolving direct-coupling architecture implicitly via Observer Pub-Sub mechanisms scaling Entity states correctly asynchronously. |
| [Lab07](Lab07/) | **Async & Await** | Mitigating Application UI blocking implicitly utilizing ThreadPool computations mapping huge native math queries logically avoiding explicit synchronization hangs optimally. |
