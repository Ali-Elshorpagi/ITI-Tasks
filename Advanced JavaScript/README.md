# Advanced JavaScript

## Course Description

This directory contains labs and resources covering concepts in **Advanced JavaScript**, including Closures, Higher-Order Functions, Prototypes, Object-Oriented Programming (OOP), and Asynchronous Programming.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **Closures & Higher-Order Functions** | Implementing closures (counters, trackers, private state) and using higher-order functions (`map`, `filter`, `reduce`). |
| [Lab02](Lab02/) | **Prototypes & OOP** | Implementing Constructor functions and Prototype-based inheritance (`Car` and `ElectricCar`), and practicing the spread operator. |
| [Lab03](Lab03/) | **Asynchronous JavaScript & APIs** | Fetching data from the REST Countries API (Promises), displaying neighbors, and building a structured CRUD application with pagination, search, and sorting. |
