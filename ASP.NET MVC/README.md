# ASP.NET MVC

## Course Description
This directory contains labs and resources covering concepts in **ASP.NET MVC** as part of the .NET back-end track. Topics progress from MVC fundamentals and routing to full-stack applications using Entity Framework Core, the Repository Pattern, ViewModels, and Razor views.

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| [Lab01](Lab01/) | **MVC Basics & Routing** | Controllers, action methods, HTTP verb attributes, `ViewBag`, form submission — builds a simple Utility calculator app (Add / Subtract) |
| [Labs](Labs/) | **Full MVC App with EF Core & Repository Pattern** | Generic `IRepository<T>`, EF Core Code-First with Migrations, full CRUD for Students / Departments / Courses, ViewModel validation, many-to-many course assignment |
